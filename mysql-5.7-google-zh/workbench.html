第30章MySQL Workbench

MySQL Workbench提供了一个用于处理MySQL服务器和数据库的图形化工具。MySQL Workbench完全支持MySQL版本5.1及以上版本。

以下讨论简要介绍了MySQL Workbench的功能。有关更多信息，请参阅http://dev.mysql.com/doc/workbench/en/上提供的MySQL Workbench手册 。

MySQL Workbench提供了五个主要的功能区域：

  • SQL开发：使您能够创建和管理与数据库服务器的连接。除了使您能够配置连接参数外，MySQL Workbench还提供了使用内置SQL编辑器在数据库连接上执行SQL查询的功能。该功能取代了以前由查询浏览器独立应用程序提供的功能。

  • 数据建模：使您能够以图形方式创建数据库架构模型，在模式和实时数据库之间进行反向和正向工程，并使用综合的表编辑器编辑数据库的所有方面。表编辑器提供易于使用的工具来编辑表，列，索引，触发器，分区，选项，插入和特权，例程和视图。

  • 服务器管理：使您能够创建和管理服务器实例。

  • 数据迁移：允许您从Microsoft SQL Server，Sybase ASE，SQLite，SQL Anywhere，PostreSQL以及其他RDBMS表，对象和数据迁移到MySQL。迁移还支持从早期版本的MySQL迁移到最新版本。

  • MySQL企业支持：支持企业产品，如MySQL企业备份和MySQL审计。

MySQL Workbench有两个版本，Community Edition和Commercial Edition。社区版是免费提供的。商业版以低成本提供其他企业功能，例如数据库文档生成。
