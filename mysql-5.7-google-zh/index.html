MySQL 5.7参考手册

包括MySQL NDB Cluster 7.5和NDB Cluster 7.6

抽象

这是MySQL™参考手册。它记录了MySQL 5.7到5.7.23以及分别基于7.5.2 NDB到5.7.22-ndb-7.5.11 版本的NDB集群版本 。

MySQL 5.7功能。  本手册介绍每个版本的MySQL 5.7中未包含的功能; 这些功能可能不包含在授权给您的MySQL 5.7版本中。如果您对您的MySQL 5.7版本中包含的功能有任何疑问，请参阅MySQL 5.7许可协议或联系您的Oracle销售代表。

有关每个版本中的更改的详细说明，请参阅 MySQL 5.7发行说明。

有关法律信息，请参阅法律声明。

有关使用MySQL的帮助，请访问 MySQL论坛或 MySQL邮件列表，您可以在其中与其他MySQL用户讨论您的问题。

有关MySQL产品的其他文档（包括将文档翻译成其他语言）以及各种格式（包括HTML和PDF格式）的可下载版本，请参阅MySQL文档库。

许可信息 - MySQL 5.7。  本产品可能包含根据许可使用的第三方软件。如果您正在使用商业 版本的MySQL 5.7，请参阅 MySQL 5.7商业版本许可信息用户手册以获取许可信息，包括与本商业版本中可能包含的第三方软件相关的许可信息。如果您使用的是 社区版本的MySQL 5.7，请参阅 MySQL 5.7社区版本许可信息用户手册用于许可信息，包括与本社区版本中可能包含的第三方软件相关的许可信息。

许可信息 - MySQL NDB Cluster 7.5。  本产品可能包含根据许可使用的第三方软件。如果您正在使用 NDB Cluster 7.5 的商业版本，请参阅 MySQL NDB Cluster 7.5商业版本许可信息用户手册，以获取与此商业版本中可能包含的第三方软件相关的许可信息。如果您使用的是NDB Cluster 7.5 的 社区版本，请参阅“ MySQL NDB Cluster 7.5社区版本许可信息用户手册” 用于许可与本社区版本中可能包含的第三方软件相关的信息。

许可信息 - MySQL NDB Cluster 7.6。  如果您使用的是NDB Cluster 7.6的开发人员预览版本，请参阅 MySQL NDB Cluster 7.6社区版本许可信息用户手册，以获取与本社区版本中可能包含的第三方软件相关的许可信息。
