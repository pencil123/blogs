Python 调用shell
admin 2016/08/12 2018-06-02

使用Python调用Shell命令有如下4种方式

1、os.system方法

os.system(“The command you want”). 这个调用相当直接，且是同步进行的，程序需要阻塞并等待返回。
返回值是依赖于系统的，直接返回系统的调用返回值，所以windows和linux是不一样的
>>> import os
>>> os.system('pwd')
/root
0
>>> os.system('ls')
1 2 anaconda-ks.cfg db install.log install.log.syslog ip python Python-3.4.4.tgz
0

2、os.popen方法

os.popen(command[,mode[,bufsize]])，popen方法通过p.read()获取终端输出，而且popen需要关闭close()。
当执行成功时，close()不返回任何值，失败时，close()返回系统返回值. 可见它获取返回值的方式和os.system不同。
>>> p = os.popen("dir /root","r")
>>> p.read()
'1 2 anaconda-ks.cfg db install.log\tinstall.log.syslog ip\tpython\tPython-3.4.4.tgz\n'
>>> p.close()

4、subprocess模块

以上三个方法都不是Python推荐的方法，而且在Python3中其中两个已经消失。
Python 推荐使用subprocess执行外部命令。
>>> import subprocess
>>> subprocess.call(["pwd"],shell=True)
/root
0

>>> subprocess.call(["date"],shell=True)
Fri Aug 12 09:48:13 MYT 2016
0

直接调用命令，返回值即是系统返回。shell=True表示命令最终在shell中运行。Python文档中出于安全考虑，不建议使用shell=True。建议使用Python库来代替shell命令。
