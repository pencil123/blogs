flock：Manage locks from shell scripts

Usage:
flock [-sxun][-w #] fd#
flock [-sxon][-w #] file [-c] command...
flock [-sxon][-w #] directory [-c] command...

-s –shared 获取共享锁 Get a shared lock
-x –exclusive 获取排它锁 Get an exclusive lock
-u –unlock 移除锁 Remove a lock
-n –nonblock 非阻塞模式，不等待 Fail rather than wait
-w –timeout 超时时间 Wait for a limited amount of time
-o –close 命令执行器关闭描述符 Close file descriptor before running command
-c –command 执行的shell命令 Run a single command string through the shell

使用场景：
当设定时间间隔计划任务时，如果某次脚本运行很慢；有可能会出现上次脚本还没跑完，这次脚本要开始跑的情况。
这样就会有出现系统同一时刻，执行多个脚本。
如果脚本书写不当，不适合多个脚本一起运行；则会产生很大问题。
所以使用flock锁机制，可以避免多个脚本一起跑的情况。
