python 脚本封装为exe文件
admin 2016/06/14 2016-10-15

作为运维工程师，自然要以支持开发，支持项目为第一重要任务。最近项目客户有反馈说网站加载慢，这个问题也是linux系统运维工程师的职责范围内的。所以怎么解决那？
我们这边测试网站访问正常，但客户那边的确有问题。于是就想着写一个脚本，检查收集一下客户的数据。
将python封装为exe文件，试过两个工具：py2exe 和PyInstaller
1、py2exe 使用过程中总是报找不到readline等模块的错误，试了几个放开没有解决，接放弃了。
这里主要记录一下PyInstaller的使用：
1、首先配置环境：在windows上安装python开发环境，添加系统变量；网上教程很多也就不赘述了。
2、去官网下载安装包，http://www.pyinstaller.org/ 我使用的下载地址为：http://pilotfiber.dl.sourceforge.net/project/pyinstaller/2.0/pyinstaller-2.0.zip
3、安装依赖PYwin：windows开发python的工具；http://tenet.dl.sourceforge.net/project/pywin32/pywin32/Build%20220/pywin32-220.win-amd64-py3.4.exe
UPX 文件压缩器：http://upx.sourceforge.net/download/upx391w.zip
将上面需要的东西安装成功就可了

使用方法：
pyInstaller 安装
Before using any contributed platform, you need to build the PyInstaller
bootloader, as we do not ship binary packages. Download PyInstaller
source, and build the bootloader::

cd bootloader
python ./waf distclean all

Then install PyInstaller::

python setup.py install

or simply use it directly from the source (pyinstaller.py).

UPX安装
下载后的文件 copy 到 PyInstaller的文件目录下

打包文件
Usage
—–

Basic usage is very simple, just run it against your main script::

pyinstaller /path/to/yourscript.py

For more details, see the `manual`_.
