elasticsearch命令

Option Description
-E <KeyValuePair> Configure a setting
-V, –version Prints elasticsearch version information and exits
-d, –daemonize Starts Elasticsearch in the background
-h, –help show help
-p, –pidfile Creates a pid file in the specified path on start
-q, –quiet Turns off standard ouput/error streams logging in console
-s, –silent show minimal output
-v, –verbose show verbose output

1、es默认配置文件目录config；如果要修改使用-Epath.conf=pathtoconf。
2、-d 后台运行，-p指定pid文件。

启动：
bin/elasticsearch -d -p pid.file
tip：es禁止使用root账号启动

停止：
An orderly shutdown of Elasticsearch ensures that Elasticsearch has a chance to cleanup and close outstanding resources.
有序的停止服务，保证了es可以清理和关闭使用的资源。

前台运行：
在终端使用Control-C组合键，正常停止es服务。

后台运行：发送SIGTERM信号
kill -TERM es-process-id

另外还有_shutdown接口，可以关闭服务。
