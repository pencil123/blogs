计划资源 Schedule

Name required
run Level Date-time-specification 设置在哪个日期时间做的作业等级

文件集资源 FileSet

FileSet资源定义要在备份作业中包含或排除哪些文件。一个文件集资源需要为每个备份作业。它由要包含的文件或目录列表，要排除的文件或目录列表以及要应用于每个文件的各种备份选项（如压缩，加密和签名）组成。
Name required
Include 包含
Exclude 排除
File 指定目录或文件名称；可以多次使用

客户端资源 Client

Name required
Address ip地址或者主机名
FDPort 客户端端口
Password 与File服务建立连接时要使用的密码。客户端配置文件的密码必须与此密码相同
File Retention 文件保留，与作业保留和卷保留一起使用
Job Retention 作业保留，删除指定文件更早的作业记录。

存储资源 Storage

存储资源定义哪些存储守护程序可供Director使用。
Job 资源定义使用的Storage ; Storage 定义使用的Device;
Storage Daemon 配置文件bacula-sd.conf 中定义Device 或者AutoChanger。

Name required
Address rIP地址或主机名
SDPort r机器端口号
Password r密码
Device required r指定存储进程(Storage daemon)使用的存储设备(device resource)。
如果使用Autochanger，则为Autochanger的名称。
Media Type r指定用于存储数据的介质类型

池资源 Pool

定义了Bacula用来写入数据的一组存储卷（磁带或文件）
意义：创建卷时，定义了文件保保留期、回收标志、存储文件数等
使用说明：
1、在Director配置文件中定义；并会写入数据库中。
2、物理介质需要贴上标签；控制台console中label的指令
3、 Pool设置属性；需要注意介质类型Media Type的设置一致性。

Name required
Pool Type 池类型，对应于正在运行的作业类型。Backup
Storage 定义数据备份写入的位置。和Job 中的Storage指令相同；不过如果Pool中也使用了Storage指令，则Pool中的指令会生效。
Maximum Volume Files 此指令指定可写入卷的最大文件数
Maximum Volume Bytes 该指令指定了可以写入卷的最大字节数

编目资源 Catalog

Name required
dbpassword 数据库密码
dbname 数据库名
dbuser 数据库用户名

消息资源 Message

控制器资源 Console

Name required
password 连接Director控制器时验证的密码
