JS函数
admin 2015/11/16 2018-01-14

Function 构造函数 创建一个新的Function对象。 在 JavaScript 中, 每个函数实际上都是一个Function对象。
它可以像任何其他对象一样具有属性和方法。它们与其他对象的区别在于函数可以被调用。

普通函数中没有使用return语句，则它默认返回undefined。
使用new关键字调用一个构造函数除外。new默认返回一个对象。

函数定义

1、函数声明 (函数语句)
function name([param[, param[, ... param]]]) { statements }

2、函数表达式 (function expression)
function [name]([param] [, param] [..., param]) { statements }

当函数只使用一次时，通常使用IIFE (Immediately Invokable Function Expressions)。
(function() {
statements
})();
IIFE是在函数声明后立即调用的函数表达式。

3、函数生成器声明 (function* 语句)

4、函数生成器表达式 (function* 表达式)

5、构造函数
一个用Function构造函数定义的函数,被赋值给变量multiply：
var multiply = new Function('x', 'y', 'return x * y');

JS解析器会将函数声明提前到整个作用域的最前面，所以可以将函数声明定义在js的任意位置。
函数表达式不会被解析器提前处理，所以函数表达式要放在使用之前。

匿名函数：没有明明的函数
1、用于绑定事件，
2、自调用函数IIFE

回调函数
简单理解就是函数做为参数。
复杂理解：回调函数就是一个通过函数调用的函数。如果你把函数的指针（地址）作为参数传递给另一个函数，当这个指针被用来调用其所指向的函数时，我们就说这是回调函数。
