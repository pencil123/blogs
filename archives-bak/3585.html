Bacula GUI Software
admin 2018/02/16 2018-03-26

配置好Bacula服务后，就需要安装一个GUI管理界面了。Bacula都有哪些GUI页面软件可以选择那？

Bacula-Web

Bacula-web是一个基于PHP的Web程序，提供已经运行的作业的汇总输出。
它从您的目录数据库中获取其信息。除了一个很好的图形显示，它提供了您的工作摘要，以及工作使用情况的图表。
Bacula-Web 不能修改Bacula服务运行信息，不能执行运行任务和删除备份操作。

Bat

The Bacula Administration Tool
功能很强大，是商业产品，需要购买。
它具有比bwx-console和bgnome-console更多的功能，并且随着时间的推移，许多附加功能将被添加，包括托盘监视器的功能以及bweb的报告功能。

bimagemgr

Bimagemgr是一个以Perl编写的基于Web的界面，用于监视要写入CDROM的磁盘卷。

bwx-console

bwx-console是一个用wxWidgets编写的图形控制台界面，可在所有客户端平台上使用。bwx-console允许您在标准tty控制台中执行任何操作，另外还有一个基于图形树的点和点击恢复功能。

bgnome-console

bgnome控制台是支持GNOME 2.x的系统上可用的图形控制台界面。

BWEB

Bweb是一个基于perl的web程序，提供了一个工具来执行基本操作并获得统计信息。（它需要Bacula> = 1.39）它从您的目录数据库和bconsole程序中获取其信息。
它的一些主要特点如下：
实时跟踪工作进展（包括客户状态和工作日志）
查看池/媒体占用
更新卷参数
管理位置（使用工作流程移入/移出媒体）
获取有关作业的统计信息（文件号，作业大小，作业持续时间）
运行一份新工作
管理您的自动换片器
适用于PostgreSQL和MySQL
在Mozilla和Firefox下正常工作

brestore

Brestore是支持Perl / GTK / Glade的系统上可用的图形恢复界面

补充说明

以上GUI 软件我配置过Bacula-Web，配置过程很简单；需要注意php的版本和扩展。
其他软件是否要用，没有发言权。

另外Bacula官网提供的Bacula-gui包说明为：
该文件包含非核心GUI程序。
目前，它包含bacula-web，一个PHP程序，用于在浏览器中管理您的Bacula作业状态; 和bimagemgr浏览器程序，用Bacula卷刻录CDROM映像。
