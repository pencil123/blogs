字符串拼接

数据库操作中会遇到将两个字段的字符内容拼接在一起使用的情况。Mysql中使用concat命令用于拼接字符串。
Sqlite则使用两个竖线(||)用来拼接。
update user set sex = "性别:"||sex where username in ("zhang","wang","li");
如果要对两个数值字段进行计算，则可以直接使用加减乘除进行操作。
