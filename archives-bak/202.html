远程同步工具Rsync
admin 2016/06/01 2018-02-21

Rsync说明

rsync(Remote Sync:远程同步)是类unix系统下的数据镜像备份工具。

工作模式

    • 本地文件镜像拷贝
    • 远程Shell(ssh|rsh)拷贝
    • 远程rsync守护进程拷贝(数据传输加密需要依赖ssh或stunnel)
    • 查询列表模式

特性

可以镜像保存整个目录树和文件系统。
可以很容易做到保持原来文件的权限、时间、软硬链接等等。
无须特殊权限即可安装。
快速：第一次同步时 rsync 会复制全部内容，但在下一次只传输修改过的文件。rsync 在传输数据的过程中可以实行压缩及解压缩操作，因此可以使用更少的带宽。
安全：可以使用scp、ssh等方式来传输文件，当然也可以通过直接的socket连接。
支持匿名传输，以方便进行网站镜象。

配置说明

守护进程启动

usr/bin/rsync --daemon --config=/etc/rsyncd/rsyncd.conf

客户端同步

rsync -参数 用户名@同步服务器的IP::rsyncd.conf中那个方括号里的内容 本地存放路径 如:
rsync -avzP nemo@192.168.10.1::nemo /backup

操作实例

服务器端安装和配置

Shell > yum install rsync
Shell > cat /etc/xinetd.d/rsync
# default: off
# description: The rsync server is a good addition to an ftp server, as it allows crc checksumming etc.
service rsync
{
disable = yes
flags = IPv6
socket_type     = stream
wait            = no
user            = root
server          = /usr/bin/rsync
server_args     = --daemon
log_on_failure  += USERID
}
配置Xinetd服务启用Rsync服务(disable=no)

配置文件

Shell > vi /etc/rsyncd.conf
#全局配置
Uid = root
Gid = root
Port = 873   #运行的默认端口
Hosts allow = 192.*.*.*   #允许访问的主机
Hosts deny = *    #拒绝访问的主机
Use chroot =          #服务器守护程序在将chroot 到文件系统中的目录中
Max connections = #客户端最多连接数
Timeout =  #超时时间
Log fromat = %t %a %m %f %b
Syslog facility = local3
Pid file = /var/run/rsyncd.pid
Lock file = /var/run/rsyncd.lock
Log file = /var/log/rsyncd.log
#motd file = /etc/rsyncd.motd #欢迎文件
#模块配置
[testmod]
Auth users = root #认证用户是root  ，是必须在服务器上存在的用户
Path = /home/rsync
Comment = rsync files
Ignore errors
Read only = no
Write only = no
List = yes    #rsync 服务器上提供同步数据的目录在服务器上模块是否显示列出来。
Secrets file = /home/rsyncd.secrets

认证文件

Shell > vim /home/rsyncd.secrets
Root:helloworld        #root账户密码helloworld
Shell >chmod 600 /home/rsyncd.secrets       #设定安全权限

客户端配置

认证文件

vim /etc/rsync.pass
helloworld
chmod 600 /etc/rsync.pass #文件安全权限

目录同步

/usr/bin/rsync -vzrtopg --delete --progress rsync@192.168.8.126::test /usr/local/share/rsync_backup --password-file=/etc/rsync.pass

上传

rsync -vzrtopg --progress /home/zhangly root@192.168.30.51::test

下载

rsync -vzrtopg --delete --progress rsync@192.168.8.126::test /home/rsync
