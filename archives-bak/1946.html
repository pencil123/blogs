postfix 基本配置
admin 2017/07/09 2017-11-02

基本配置

myhostname = 主机名#默认情况下，其值被设定为本地机器名；

myorigin = 出站邮件中使用的域名

mydomain = 使用的域名；myhostname的域名部分

mydestination = 本机接受这些域名的邮件

mynetworks = 信任的网络地址，提供中继服务

relay_domains = 那些地址域名提供中继

inet_interfaces = 监听的网络接口

mail_owner = postfix

增强版

local_destination_concurrency_limit = 2 并发连接设置
default_destination_concurrency_limit = 10 并发连接设置

smtpd_banner = #设置向用户显示的主机名和版本信息

notify_classes = #向postfix管理员报告的信息量

smtpd_sasl_auth_enable = #服务器使用SMTP认证

smtpd_sasl_local_domain = #服务器使用SMTP认证

smtpd_sasl_security_options = #不允许匿名的方式认证

message_size_limit = 15728640#规定邮件最大尺寸为15MB

用户别名

1、在配置文件开启基于hash的别名文件支持

在main.cf中，找到如下指令，而后启用它(即移除前面的#号)：
#alias_maps = hash:/etc/aliases

2、在/etc/aliases文件中定义新的别名项，其格式通常为以冒号隔开的两个字段，前一个字段为初始目标邮件地址，后一个字段为实际发往的地址，如：
redhat： magedu
gentoo@126.com: admin@magedu.com

3、将/etc/aliases转换为hash格式：
# postalias /etc/aliases
