JS几种对象创建方法
admin 2015/11/27 2018-01-14

使用new关键字创建
new一个对象;然后再为对象添加属性和方法

var student = new Object();
stu.name = "mayun";
stu.age = 20;
stu.action = function() {
console.log(this.name + ":good good study");
}

使用字面量创建
定义，声明，直接实例化

var stu = {
name : "mayun",
age : 20,
action : function() {
console.log(this.name + ":good good study");
}
}

工厂模式
如果一个类（对象模板）要实例化很多对象，需要使用工厂模式。

function createstu(name, age) {
var o = new Object();
o.name = name;
o.age = age;
o.action = function() {
alert(this.name + ":good good study");
}
return o;
}
var stu1 = createstu("hanmeimei",12);
var stu2 = createstu("ligoudan",12);

构造函数

function stu(name,age){
this.name = name;
this.age = age;
this.action = function(){
alert(this.name + ":good good study");
}
}
var stu1 = new stu("vivian",13);
var stu2 = new stu("vivian2",13);

在这里我们创建一个新对象，然后将构造函数的作用域赋给新对象，调用构造函数中的方法。

使用以上的方法创建对象，会发现两个实例中的action方法不是同一个Function的实例。
console.log(stu1.action == stu2.action); //false
为了使代码更合理，可以使实例引用同一个Function实例。

优化action函数放到构造函数外面声明：

function stu(name,age){
this.name = name;
this.age = age;
this.action = action
}
function action(){
alert(this.name + ":good good study");
}
