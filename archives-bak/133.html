postfix
admin 2017/07/09 2017-11-02

安装
1、Yum remove sendmail
#如果存在，卸载sendmail

2、Alternatives –config mta /usr/sbin/sendmail.postfix
#更改系统默认的mta服务

3、Yum install postifx

配置文件
master.cf: 主进程配置文件
main.cf:整个邮件的配置文件

main配置文件

soft bounce: 软边界
local pathname information: 本地目录信息
queue and process ownership: 队列和进程所有权
interent host and domain names： 网络主机和域名
sending mail: 发送邮件
receiving mail： 接受邮件
network:ipv4， 代理
rejecting mail for unknown local users： 拒绝未知本地用户的邮件
trust and delay contral： 信任和延迟控制
internet or intranet： 互联网或内部网
rejecting unknown relay users： 拒绝未知中继用户
input rate control： 输入速率控制
address rewriting： 地址重写
address redirection： 地址重定向
alias database： 别名数据库
delivery to mailbox： 传递到邮箱
junk mail controls： 垃圾邮件控制

注意：
1、在postfix的配置文件中，参数行和注释行是不能处在同一行中的；
2、任何一个参数的值都不需要加引号，否则，引号将会被当作参数值的一部分来使用；
3、每修改参数及其值后执行 postfix reload 即可令其生效；但若修改了inet_interfaces，则需重新启动postfix；
4、如果一个参数的值有多个，可以将它们放在不同的行中，只需要在其后的每个行前多置一个空格即可；postfix会把第一个字符为空格或tab的文本行视为上一行的延续；

说明：
postfix安装成功后，限制谁可以发邮件的方式：
1、基于mynetworks设置可信任的网络，只接受信任网络的邮件。
2、扩展SASL服务，进行用户登录认证。
如果没有扩展SASL服务的话，postfix也可以对系统用户进行验证；不过不能禁止匿名用户发邮件，也能禁止A用户冒充B用户发邮件。所以需要扩展SASL。

注意：
postfix监听25号端口，有时候运营商对25号端口传输的数据进行丢失。
