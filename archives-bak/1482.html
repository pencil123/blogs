导出数据

导出csv格式，需要指定字段，使用逗号分隔
mongoexport -h 主机ip –port 端口 -u username -p password -d database -c collection -f fields –type=csv|json -o outputfile

备份数据

mongodump -h 主机 -d 数据库 -c 集合 -o dumpdir
将指定数据备份到指定目录；
目录结构：database/数据文件； 每个集合有两个数据文件：collection.bson存储数据 和colletion.metadata.json 存储结构

还原数据

mongorestore -h ：指定还原的数据目录位置；和mongodump命令中的dumpdir相同；会根据目录里不同数据库的内容恢复。

批量执行

在处理mongodb数据时，如果存在多条命令批量操作；那么编辑一个命令文档，然后将名单一次性执行完成。这样会方便很多。

方法1：
bin/mongo serverhost:serverpost/CollectionName command_file
指明ip:port 和集合

方法2：
在mongdb shell中切换到使用的集合
使用load(“/command_file”)命令
