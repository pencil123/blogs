命令结合工具 xargs
admin 2015/09/21 2018-02-12

build and execute command lines from standard input
从标准输入中构建和执行命令行

说明

xargs 从标注输入读取items，使用’空格’或’分隔符’分割；组合到后续的命令，用作参数。

部分参数说明

-0, –null 将stdin中的特殊字符(空格\换行)，按普通字符处理。
-a file, –arg-file=file 从文件中读入，而不是从stdin
–delimiter=delim, -d delim 分隔符
-E eof-str 结束符
-e[eof-str], –eof[=eof-str] Set the end of file string to eof-str.
-I replace-str 用初始参数中的replace-str替换从标准输入中读取的名称。
-i[replace-str], –replace[=replace-str]
-L max-lines 每个命令行最多使用最多行非空白输入行。
-n max-args, –max-args=max-args 命令做大args的个数
-P max-procs, –max-procs=max-procs 同时最大的进程数，default 1
-o, –open-tty Reopen stdin as /dev/tty in the child process before executing the command.
-p, –interactive 交互，执行时提醒
–process-slot-var=name 环境变量名称中设置一个唯一值运行子进程
-r, –no-run-if-empty 当xargs的输入为空的时候则停止xargs，不用再去执行了
-s max-chars, –max-chars=max-chars 每个命令行最多使用max-chars字符，包括命令和初始参数以及终止零点参数字符串的结尾。
–show-limits 显示命令行长度的限制

示例

1、单行输出
cat /etc/passwd |xargs

2、设定每行参数个数
cat /etc/passwd |xargs -d: -n2

3、设定字符串用于替换为标准输入的内容
username 为替换字符串
cat /etc/passwd |cut -d: -f1 |xargs -I username echo "hello" username
hello root
hello bin
