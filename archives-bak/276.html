vmstat 虚拟内存统计
admin 2015/09/15 2018-01-17

vmstat : Report virtual memory statistics 报告虚拟内存统计信息

vmstat 虽然字面意思是虚拟内存使用统计检查，不过实际功能包含了进程、内存、虚拟内存、I/O、系统和CPU运行信息。

Usage:vmstat [-a] [-n] [-t] [-S unit] [delay [ count]]
delay :设置时间间隔
count :设置展示的次数

输出信息说明

Procs 进程
r: 分配到CPU,运行队列中的进程数
b: Sleep状态的进程数

Memory 内存
swpd: 虚拟内存使用SIZE
free: 空闲内存SIZE
buff: buffers 缓冲使用的内存SIZE
cachene: cache 缓存使用的内存SIZE
inact: 不活动的内存SIZE (-a option)
active: 活动内存SIZE (-a option)

Swap 交换分区
si: 从硬盘的交换分区(SWAP)写入到内存的size/second.
so: 从内存中写入到硬盘的交换分区的size/second.

IO 写入/读出（block块的size）
bi: 从块设备读取块的数量SIZE/second.
bo: 写入到块设备的块的数量SIZE/second.

System 系统
in: 每秒中断的数量，包括时钟
cs: 每秒的上下文切换次数。

CPU 这里是CPU总时间的百分比
us: 用户空间占用百分比
sy: 内核空间占用百分比
id: 时间闲置。在Linux 2.5.41之前，这包括IO等待时间。
wa: 等待IO的时间。在Linux 2.5.41之前，包含在闲置中。
st: 从虚拟机盗取的时间。在Linux 2.6.11之前，未知。
