Python 爬虫模块库说明-PycURL
admin 2016/07/28 2018-05-28

Python从Internet抓取信息的库有PycURL和urllib。

PycURL 模块说明

A Python Interface To The cURL library（C语言编写的库）;官网：https://curl.haxx.se/libcurl/
PycURL是libcurl（多协议文件传输库）的Python接口。 与urllib Python模块类似，PycURL可用于从Python程序中获取由URL标识的对象。
然而，除了简单的提取之外，PycURL还保留了libcurl的大部分功能，包括：
1、速度 – libcurl速度非常快（参考多并发操作），而PycURL作为libcurl之上的薄包装，速度也非常快。 PycURL的基准测试比Requests(urllib)要快好几倍。
2、功能包括多个协议支持，SSL，身份验证和代理选项。 PycURL支持大部分libcurl的回调。
3、众多的和共享的接口,较为复杂。
4、用于网络操作的套接字，允许将PycURL集成到应用程序的I/O循环中（例如使用Tornado）。
安装
pip install pycurl

PycURL 模块常用方法

pycurl参数传递方法，以及基于pycurl模块实现的一个web页面探测程序。

pycurl.Curl()类 实现一个ibcurl包的Curl句柄对象
close() 对应libcurl包中curl_easy_cleanup方法，无参数，实现关闭回收Curl对象
perform() 对应libcurl包中curl_easy_perform方法，无参数，实现Curl对象请求的提交
setopt(option,value) 对应libcurl包中curl_easy_setopt方法
参数option通过libcurl的常量来定义，参数value的值会依赖option，可以是一个字符串、整形、长整形、文件对象、列表、函数等

curl_easy_setopt 方法说明

1、函数通过对参数的设置，控制libcurl的行为。
2、选项通过参数设置；参数可为长，支持，或者…
3、认真阅读文档，错误的参数会导致错误的行为。
4、可以对函数设置一个参数或者多个参数
5、使用此函数调用设置的选项对于使用此句柄执行的所有即将传输都有效。
6、这些选项在传输之间没有任何重置，所以如果您希望后续传输具有不同的选项，则必须在传输之间进行更改。 您可以选择使用curl_easy_reset将所有选项重置为内部默认值。

选项分类

behavior options 行为选项 ftp options FTP选项
callback options 回拨选项 rtsp options RTSP OPTIONS
error options 错误选项 protocol options 协议选项
network options 网络选项 connection options 连接选项
names and passwords options (authentication) 名和密码选项（验证） ssl and security options SSL和安全选项
http options HTTP OPTIONS ssh options SSH选项
smtp options SMTP选项 other options 其他选项
tftp options TFTP选项 telnet options TELNET选项

设置选项

>>> import pycurl #模块导入
>>> curl = pycurl.Curl() #创建curl对象
>>> curl.setopt(pycurl.CONNECTTIMEOUT,5) #连接等待时间，0则不等待
>>> curl.setopt(pycurl.ENCODING, ‘gzip’) #Accept-Encoding and automatic decompressing data
>>> curl.setopt(pycurl.REFERER,’http://www.google.com/search?sourceid=chrome&ie=UTF-8&q=’+self.randstr()) #设置header请求来源地址
>>> curl.setopt((pycurl.HTTPHEADER,[‘text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8’])
>>> curl.setopt(pycurl.COOKIEFILE, “cookie_file_name”) #保持连接，写cookie文件
>>> curl.setopt(pycurl.TIMEOUT,5) #超时时间
>>> curl.setopt(pycurl.NOPROGRESS,0) #下载进度条，非0则屏蔽
>>> curl.setopt(pycurl.MAXREDIRS,5) #指定HTTP重定向最大次数
>>> curl.setopt(pycurl.FOLLOWLOCATION, 1) #允许跟踪来源
>>> curl.setopt(pycurl.FORBID_REUSE,1) #完成交互后强制断开连接，不重用
>>> curl.setopt(pycurl.DNS_CACHE_TIMEOUT,60) #设置DNS信息保存时间，默认为120秒
>>> curl.setopt(pycurl.USERAGENT, “Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/51.0.2704.103 Safari/537.36″) #设置HTTP的User-Agent（自行设置时需跟着常规标准走）
>>> curl.setopt(pycurl.URL,”http://www.plcxue.com”) #设置请求的Url
>>> curl.setopt(pycurl.WRITEFUNCTION,buf.write) #回调 和下面的参数向重合；不要同时使用
>>> curl.setopt(pycurl.HEADERFUNCTION,getheader) 将返回的HTTP HEADER定向到回调函数getheader
>>> curl.setopt(pycurl.WRITEHEADERFUNCTION,getbody) #将返回的内容定向到回调函数getbody
>>> curl.setopt(pycurl.WRITEHEADER,fileobj) #将返回的HTTP HEADER定向到fileobj文件对象
>>> curl.setopt(pycurl.WRITEDATE,fileobj) #将返回的HTML内容定向到fileobj文件对象
>>> curl.setopt(pycurl.NOSIGNAL, 1) #

获取结果信息

对应libcurl的curl_easy_getinfo方法，参数option通过libcurl的常量指定。
>>> curl = pycurl.Curl()
>>> curl.getinfo(pycurl.HTTP_CODE) #返回HTTP状态码
>>> curl.getinfo(pycurl.TOTAL_TIME) #传输结束时所消耗的总时间
>>> curl.getinfo(pycurl.NAMELOOKUP_TIME) #DNS解析所消耗的时间
>>> curl.getinfo(pycurl.CONNECT_TIME) #建立连接所消耗的时间
>>> curl.getinfo(pycurl.PRETRANSFER_TIME) #从建立连接到准备传输所消耗的时间
>>> curl.getinfo(pycurl.STARTTRANSFER_TIME) #从建立连接到数据开始传输所消耗的时间
>>> curl.getinfo(pycurl.REDIRECT_TIME) #重定向所消耗的时间
>>> curl.getinfo(pycurl.SIZE_UPLOAD) #上传数据包大小
>>> curl.getinfo(pycurl.SIZE_DOWNLOAD) #下载数据包大小
>>> curl.getinfo(pycurl.SPEED_DOWNLOAD) #平均下载速度
>>> curl.getinfo(pycurl.SPEED_UPLOAD) #平均上传速度
>>> curl.getinfo(pycurl.HEADER_SIZE) #HTTP头部大小

_Curl.setopt(pycurl.MAXREDIRS,1)
_Curl.setopt(pycurl.DNS_CACHE_TIMEOUT,30)
_Curl.setopt(pycurl.URL,URL)
try:
with open(os.path.dirname(os.path.realpath(__file__)) + “/content.txt”,’w’) as outfile:
_Curl.setopt(pycurl.WRITEHEADER,outfile)
_Curl.setopt(pycurl.WRITEDATA,outfile)
#Here we write the body to a buffer to avoid printing uninteresting output to standard out.
_Curl.perform()
except Exception as err:
print “exec error!\n\t%s” %err
sys.exit()
print “Http Code:\t%s” %_Curl.getinfo(_Curl.HTTP_CODE)
print “DNS lookup time:\t%s ms” %(_Curl.getinfo(_Curl.NAMELOOKUP_TIME) * 1000)

print “Ready conn time:\t%s ms” %(_Curl.getinfo(_Curl.PRETRANSFER_TIME) * 1000)
print “Tran Star time:\t%s ms” %(_Curl.getinfo(_Curl.STARTTRANSFER_TIME) * 1000)
print “Tran Over time:\t%s ms” %(_Curl.getinfo(_Curl.TOTAL_TIME) * 1000)
print “Download size:\t%d bytes/s” %_Curl.getinfo(_Curl.SIZE_DOWNLOAD)
