MongoEngine入门
admin 2017/09/13 2018-03-29

MongoEngine is a Document-Object Mapper (think ORM, but for document databases) for working with MongoDB from Python.
是一个文档对象映射框架（就像ORM，不过是为文档性数据库）；它是使用python语言对MongoDB数据库开发的。
使用文档：http://docs.mongoengine.org/

安装

pip install mongoengine

连接数据库

import mongoengine as mongo
mongo.connect('database', host='localhost', port=27017)

定义模式

文件模式继承于Document类；并通过添加字段对象（StringField，IntField等）作为Document类的属性添加到实例化的文档模式中。
class tableName(mongo.Document):
meta = {'collection': 'tableName'}
filedName1 = mongo.DateTimeField()
filedName2 = mongo.ListField(mongo.StringField())

动态文档模式

MongoDB的优点之一是集合collection的动态模式，有一些需要动态/扩展样式文档的场景。

字段类型

  • BinaryField：二进制
  • BooleanField：布尔型
  • ComplexDateTimeField
  • DateTimeField：日期
  • DecimalField：整数
  • DictField：字典
  • DynamicField
  • EmailField：邮件
  • EmbeddedDocumentField
  • EmbeddedDocumentListField
  • FileField：文件
  • FloatField：浮点
  • GenericEmbeddedDocumentField
  • GenericReferenceField
  • GeoPointField
  • ImageField：图片
  • IntField：整型
  • ListField：列表
  • MapField
  • ObjectIdField
  • ReferenceField
  • SequenceField
  • SortedListField
  • StringField
  • URLField:链接
  • UUIDField
  • PointField
  • LineStringField
  • PolygonField
  • MultiPointField
  • MultiLineStringField
  • MultiPolygonField

字段参数

  • 强制性：required=True
  • 约束性：max_length
  • 默认值：default
  • 唯一性：unique
  • 主键：primary_key
  • 可选择迭代：choices
  • 其他:unique_with

嵌入文档

EmbeddedDocument

文档集合collections

文档模式中可以设置meta参数，用于设置除了字段外的其他属性；
集合名称：collection
文档最大数：max_documents
集合最大存储：max_size

索引

在meta参数中设置indexes；
$文档索引
#哈希索引

Set any default index options - see the full options list
index_background (Optional)
Set the default value for if an index should be indexed in the background
index_cls (Optional)
A way to turn off a specific index for _cls.
index_drop_dups (Optional)
Set the default value for if an index should drop duplicates

排序

order_by()

分片key

meta = {‘shard_key’:(‘machine’,’timestamp’)}

文档继承

类似于对象开发中的对象继承
# Stored in a collection named 'page'
class Page(Document):
title = StringField(max_length=200, required=True)
meta = {'allow_inheritance': True}
# Also stored in the collection named 'page'
class DatedPage(Page):
date = DateTimeField()
