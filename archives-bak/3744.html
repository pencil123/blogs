wordpress 友情链接管理
admin 2018/03/12 2018-03-26

后台‘链接’管理菜单

wordpress 本身有管理友情链接的功能模块。如果你在后台中找不到”链接”这个管理菜单，应该是你使用的模板没有‘激活’这个功能。
在functons.php文件中，添加过滤器如下：
add_filter( ‘pre_option_link_manager_enabled’, ‘__return_true’);
刷新后台，你就能看到‘链接’管理菜单了。

前台调用链接函数

在前台调用添加的友情链接使用到的函数：wp_list_bookmarks($array_args);

categorize boolean 分类信息是否显示
category string 要显示的类别IDs,以逗号分开
exclude_category string 排除显示的IDs
category_name string 显示的类别名称
category_before string 每个类别之前放置的文本
category_after string 每个类别之后放置的文本
class string 每个类别li将有的类
category_orderby string 排序依据项
category_order string 排序顺序
title_li string 链接列表标题的文本
title_before string 每个类别描述之前放置的文本
title_after string 每个类别描述之后放置的文本
show_private string 类别被认为是私人的
include string 逗号分隔的数字书签ID列表包含在输出中
exclude string 要排除的数字书签ID的逗号分隔列表
orderby string 对书签进行排序的
order string 书签排序顺序
limit integer 最大显示数量
before string 在每个书签之前放置的文本
after string 在每个书签之后的文本
link_before string 放置在超链接代码内的每个书签的文本之前的文
link_after string 放置在每个书签文本之后的文本
between string 放置在每个书签/图像及其描述之间的文本
show_images boolean 显示书签图像
show_description boolean 显示书签描述
show_name boolean 显示链接的文本
show_rating boolean 显示评分
show_updated boolean 显示更新时间
hide_invisible boolean 是否可见
echo boolean 显示书签
