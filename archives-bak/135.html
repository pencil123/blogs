cat 命令-连接文件并打印到标准输出
admin 2016/05/23 2018-01-12

Usage: cat [OPTION]… [FILE]…
Concatenate FILE(s), or standard input, to standard output.
连接文件，或者标准输入，打印到标准输出

-A, –show-all equivalent to -vET
-b, –number-nonblank 输出非空行的行号
-e equivalent to -vE
-E, –show-ends 行尾以$标识
-n, –number 输出行号
-s, –squeeze-blank 抑制重复空行输出
-t equivalent to -vT
-T, –show-tabs display TAB characters as ^I
-u (ignored)
-v, –show-nonprinting 显示非打印;使用^和M-符号；排除LFD and TAB

dos2unix命令用来将DOS格式的文本文件转换成UNIX格式的。
DOS下的文本文件是以\r\n作为断行标志的，表示成十六进制就是0D 0A。
而Unix下的文本文件是以\n作为断行标志的，表示成十六进制就是 0A。

标准输入的内容重定向到文件中
linux:/tmp$ cat > test << EOF
> this is first line
> EOF
linux:/tmp$ cat test
this is first line
覆盖test文件，如果不存在则创建

linux:/tmp$ cat >> test << EOF
> this is five line
> this is sixth line
> EOF
追加test文件
