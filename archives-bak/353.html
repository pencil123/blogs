判断语句

编程语句中的判断语句if,else。

Python 编程语言

Python编程语句中的判断语句关键词有：if elif else。

False 的value有哪些？

  • None
  • False
  • 数值为0；zero of any numeric type,for example:0、0L、0.0、0j
  • 序列为空；any empty sequence ,for example: “”、()、[]
  • 映射为空；any empty mapping,for example:{}
  • 实例存在两个特殊方法；instances of user-defined classes,if the class defines a __nonzero__() or __Len__ method, when that method returns the integer zero or bool vlue False。

判断条件

比较运算符：>（大于）、<(小于)、==（等于）、！= （不等于）、>=（大于等于）、<=（小于等于）
同一性：x is y x和y是同一对象；is not 和 is是同一性，不是相等性
容器成员：x in y x是y容器的成员；not in 不是容器的成员

多条件判断

python 并不支持 switch 语句，所以多个条件判断，只能用 elif 来实现。

Or 和 And 操作符

x or y
执行过程：if x is false,then y,else x
说明：如果x为false；则判断y。否则不需要对y进行判断。

x and y
执行过程：if x is false,then x,else y
说明：如果x为false,则根据x值返回。否则(x为true),判断y值。

判断NoneType

判断数值是否为NoneType：
if variable is None
if x is not None

三元运算符

三元运算符是软件编程中的一个固定格式，使用这个算法可以使调用数据时逐级筛选。
语法：条件表达式？表达式1：表达式2
说明：问号前面的位置是判断的条件，判断结果为bool型，为true时调用表达式1，为false时调用表达式2。
其逻辑为：“如果为条件成立或者满足则执行表达式1，否则执行第二个。”
常用在设置默认值,例如某个值不一定存在,则判断这个值是否存在,不存在给默认值(表达式2)

循环语句

在不少实际问题中有许多具有规律性的重复操作，因此在程序中就需要重复执行某些语句。
循环语句是由循环体及循环的终止条件两部分组成的。
一组被重复执行的语句称之为循环体，能否继续重复，决定循环的终止条件。循环结构是在一定条件下反复执行某段程序的流程结构，被反复执行的程序被称为循环体。

循环结构

for循环语句
while循环语句
do while循环语句

跳出循环

break:结束循环
continue：结束本次循环
else：如果循环正常执行完（没有break），则执行(Python)

Python 语句示例

[x*x for x in range(10)]
0-9的平方

[x*x for x in range(10) if x%3 == 0]
平方值能被3整除

[(x,y) for x in range(3) for y in range(3)]
0-2:0-2的组合
