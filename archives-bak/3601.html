wordpress 列表页开发
admin 2018/02/19 2018-03-26

WordPress 列表页中主要包括的功能和元素：文章遍历显示、显示标题、固定链接、发布日期、修改日期等

代码示例

$paged = (get_query_var('paged')) ? get_query_var('paged') : 1;
query_posts( 'posts_per_page=20&paged='.$paged );
while ( have_posts() ):
the_post();
echo get_the_titile();
echo get_permalink();
echo get_the_date('Y/m/d');
echo get_the_modified_time();
endwhile;

函数说明

get_query_var() 获取当前查询的变量

检索全局WP_query对象中的wp_query类中的公共查询变量(public query variable)。
直接受已定义的变量
参考：
https://codex.wordpress.org/Class_Reference/WP_Query

query_posts() 使用查询参数设置循环

此功能将完全覆盖主要查询，并不适用于插件或主题。
它过于简单的方法来修改主查询可能有问题，应尽可能避免。

have_posts（） 判断是下一篇文章

调用全局变量$wp_query->have_posts()成员函数，来简单检查一个全局数组（array）变量$posts的一个循环计数器，以确认是否还有post。
如果有返回true（1），如果没有返回false（0）。
该函数常用于循环中。

the_post() 获取当前文章的信息

调用$wp_query->the_post()成员函数前移循环计数器，并且创建一个全局变量$post(不是$posts)。
把当前的post的所有信息都填进这个$post变量中，以备接下来使用。

get_the_titile() 获取指定/当前文章的标题

如果文章被标识为保护或隐私，则只展示给admin。

get_permalink() 获取指定/当前文章的固定链接

该函数返回文章或页面的固定链接

get_the_date() 获取当前文章的发布时间

get_the_author() 获取当前文章的作者

get_the_modified_time() 获取当前文章的最后修改时间
