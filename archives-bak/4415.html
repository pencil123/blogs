PHP foreach详解
uptoyou 2018/05/11 2018-05-15

一个普通的foreach
$arr = array(‘a’,’b’,’c’);
foreach ($arr as $key=> $value) {
echo $key, $value;
}
在foreach遍历过程中，并不是直接操作原数组（$arr），而是将原数组（$arr）复制一份出来（为方便描述暂命名为$arrcopy）（此时$arr和$arrcopy共用同一块内存），foreach在遍历过程中，操作的其实是$arrcopy
// 伪代码
$arr = array(‘a’,’b’,’c’);
/* foreach循环开始 */
// 第一次循环开始
$arrCopy = $arr; //复制出一个待循环数组的副本，接下来都是操作这个副本
$key = currentKey($arrCopy); //将获取到的值分配给$k;
$val = currentVal($arrCopy); //将获取到的值分配给$v;
next($arrCopy);//移动副本数组的指针
$arr = $arrCopy;//将副本赋值回给$arr（（主要是将指针同步移动））
// 大括号内容
{
echo $key,$value,’
’;
}
// 第一次循环结束

// 第二次循环开始
$key = currentKey($arrCopy); //将获取到的值分配给$k;
$val = currentVal($arrCopy); //将获取到的值分配给$v;
………
// 第二次循环结束
总结：foreach遍历的时候操作的并不是原始数组，而是一个拷贝数组
$arr = array(‘a’,’b’,’c’);
foreach ($arr as $key=> $value) {
$arr[] = ‘d’;
print_r($arr);
var_dump($key,$value);
}

每次循环原数组（$arr）都会增加一个值d（$arr内容改变，发生写时复制，$arr和$arrcopy已分裂，不再共用同一块内存），但foreach只遍历到c就结束了，说明在foreach遍历的是原数组（$arr）的拷贝（$arrcopy），并且该拷贝在foreach第一次遍历前就已经产生
