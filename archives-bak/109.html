Samba 文件共享说明
admin 2016/05/18 2018-05-07

Samba是在Linux和UNIX系统上实现SMB协议的一个免费软件，由服务器及客户端程序构成。
SMB（Server Messages Block，信息服务块）是一种在局域网（domain）上共享文件和打印机的一种通信协议，它为局域网内的不同计算机之间提供文件及打印机等资源的共享服务。
SMB协议是客户机/服务器型协议，客户机通过该协议可以访问服务器上的共享文件系统、打印机及其他资源。通过设置“NetBIOS over TCP/IP”使得Samba不但能与局域网络主机分享资源，还能与全世界的电脑分享资源。

RPM安装

Samba 服务在不同的Linxu 发行版中包的名称不同。以Red hat 为例按照包：

samba

这个套件主要包含了 SAMBA 的主要 daemon档案 ( smbd 及 nmbd )、 SAMBA 的文件档 ( document )、以及其它与 SAMBA 相关的logrotate 设定文件及开机预设选项档案等

samba-common

这个套件则主要提供了 SAMBA 的主要设定档(smb.conf) 、 smb.conf 语法检验的测试程序 ( testparm )等等；

samba-client

这个套件则提供了当 Linux 做为SAMBA Client 端时，所需要的工具指令，例如挂载 SAMBA 档案格式的执行档 smbmount等等。

Samba 服务

Nmbd服务主要处理netbios；
Smbd服务主要处理share文件共享。

Samba 端口

1）Port 137 (UDP) – NetBIOS 名字服务 ； nmbd
2）Port 138 (UDP) – NetBIOS 数据报服务
3）Port 139 (TCP) – 文件和打印共享 ； smbd （基于SMB(Server Message Block)协议，主要在局域网中使用，文件共享协议）
4）Port 389 (TCP) – 用于 LDAP (Active Directory Mode)
5）Port 445 (TCP) – NetBIOS服务在windos 2000及以后版本使用此端口, (Common Internet File System，CIFS，它是SMB协议扩展到Internet后，实现Internet文件共享)
6）Port 901 (TCP) – 用于 SWAT，用于网页管理Samba

认证方式

Samba认证方式分为：share，user，server，domain。

Share 认证方式

不需要输入用户名和密码；配置很方便。需要注意对共享权限和系统权限的设置。
共享权限：smb.conf文件中配置的访问权限；
系统权限：共享的目录（path）的权限；属主属组和读写；
可以通过setfacl对用户的系统权限设置。

User 认证方式

需要输入用户名，密码。Samba对用户进行认证。
User认证方式下用户管理；用户为系统用户，然后加入到samba服务用户中，同时设置samba独立用户密码。
在共享配置中设置用户的权限就可以了。同时需要注意系统权限。当系统权限和共享权限不同时，会取交集。

server 认证方式

认证工作由另一台Samba服务器或者Windows server负责

domain认证方式

这个安全级别要求网络上存在一台Windows的主域控制器，samba把用户名和密码递交给它去验证。

Samba加密口令

客户机访问Samba服务器时进行的用户身份认证包括使用”普通文本”格式的口令，通过网络传送给SMB服务器，与UNIX的帐户数据库（如/etc/passwd）进行核查；或者使用加密口令，通过”SMB
challenge/response”验证进行核查。

管理工具

testparm 检查配置文件

当设定完Samba配置文件后，使用testparm工具检测配置文件是否存在错误。

smbpasswd 管理用户

管理Samba服务的认证用户工具；常用命令：
smbpasswd -a 增加用户（要增加的用户必须以是系统用户）
smbpasswd -d 冻结用户
smbpasswd -e 恢复用户，解冻用户，让冻结的用户可以在使用
smbpasswd -n 把用户的密码设置成空；要在global中写入 null passwords -true
smbpasswd -x 删除用户

smbstatus 服务运行状态

使用smbstatus 工具查看Samba运行的状态。

smbclient 登录诊断

smbclient是访问SMB服务器资源的客户程序。
该程序提供的接口与ftp程序类似，访问操作包括从SMB服务器下载文件到本地，或从本地上载文件到SMB服务器，还可以在SMB服务器上检索目录信息等。
命令语法如下：/usr/bin/smbclient //smbserver/service [passwd][-U username]

smbmount 挂载目录

可以使用mount挂载工具替代

nmblookup

nmblookup命令用于把一个NetBIOS名字映射到IP地址。

smbprint

smbprint是在Linux系统下将打印文件提交给远地SMB服务器打印的脚本程序。
