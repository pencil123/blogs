Python 字典（Dictionary）
admin 2016/11/01 2018-05-18

字典类型

由“键-值”对组成的集合，字典中的“值”通过“键”来引用。字典的”键”是区分大小写的,也可以使用数字。
由于字典是无序的，因此字典中没有append()、remove()等方法。

字典内置函数

cmp(dict1, dict2) 比较两个字典元素。
len(dict) 计算字典元素个数，即键的总数。
str(dict) 输出字典可打印的字符串表示。
type(variable) 返回输入的变量类型，如果变量是字典就返回字典类型。

字典内置方法

dict.clear() 删除字典内所有元素
dict.copy() 返回一个字典的浅复制
dict.fromkeys(seq[, val])) 创建一个新字典，以序列 seq 中元素做字典的键，val 为字典所有键对应的初始值
dict.get(key, default=None) 返回指定键的值，如果值不在字典中返回default值
dict.has_key(key) 如果键在字典dict里返回true，否则返回false
dict.items() 以列表返回可遍历的(键, 值) 元组数组
dict.iteritems() 返回一个迭代器；和items相比，内存使用小，稍微慢
dict.keys() 以列表返回一个字典所有的键
dict.iterkeys() 返回针对键的迭代器
dict.setdefault(key, default=None) 和get()类似, 但如果键不存在于字典中，将会添加键并将值设为default
dict.update(dict2) 把字典dict2的键/值对更新到dict里
dict.values() 以列表返回字典中的所有值
dict.itervalues() 返回值的迭代器
pop(key[,default]) 删除字典给定键 key 所对应的值，返回值为被删除的值。key值必须给出。 否则，返回default值。
popitem() 随机返回并删除字典中的一对键和值。

遍历

dict={"a":"apple","b":"banana","o":"orange"}
1、通过字典的key
for i in dict:
print "dict[%s]=" % i,dict[i]
2、获取key，value列表
for (k,v) in dict.items():
print "dict[%s]=" % k,v
3、返回迭代器
for k,v in dict.iteritems():
print "dict[%s]=" % k,v
4、key，value迭代器
for k,v in zip(dict.iterkeys(),dict.itervalues()):
print "dict[%s]=" % k,v

字典排列:value值排序

dic = {'a':31, 'bc':5, 'c':3, 'asd':4, 'aa':74, 'd':0}
dict= sorted(dic.iteritems(), key=lambda d:d[1], reverse = True)
print dict
输出的结果：
[('aa', 74), ('a', 31), ('bc', 5), ('asd', 4), ('c', 3), ('d', 0)]

分解代码:
dic.iteritems()返回key,value的迭代器；
sorted方法，通过key排序；通过lambda,key等于d[1]即value。
reverse = True表示是需要翻转的，默认是从小到大，翻转的话，那就是从大到小。

字典排列:key键排序

dic = {'a':31, 'bc':5, 'c':3, 'asd':4, 'aa':74, 'd':0}
dict= sorted(dic.iteritems(), key=lambda d:d[0])# d[0]表示字典的键
print dict
