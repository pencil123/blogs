RPM 包管理工具

RedHat Package Manager

常用参数

-i, –install install package(s)
-v, –verbose provide more detailed output
-h, –hash print hash marks as package installs (good with -v)
-e, –erase erase (uninstall) package
-U, –upgrade=+ upgrade package(s)
–replacepkge 无论软件包是否已被安装，都强行安装软件包
–test 安装测试，并不实际安装
–nodeps 忽略软件包的依赖关系强行安装
–force 忽略软件包及文件的冲突

Query options (with -q or –query)

-a, –all query/verify all packages
-p, –package query/verify a package file
-l, –list list files in package
-d, –docfiles list all documentation files
-f, –file query/verify package(s) owning file
-q, –query

常用组合命令

-ivh 安装显示安装进度–install–verbose–hash
-Uvh 升级软件包–Update；
-qpl 列出指定的RPM软件包内的文件信息[Query Package list]；
-ql 通过rpm软件名，列出包文件列表
-qpi 列出RPM软件包的描述信息[Query Package install package(s)]；
-qf 查找指定文件属于哪个RPM软件包[Query File]；
-Va 校验所有的RPM软件包，查找丢失的文件[View Lost]；
-e 删除包

Yum 包管理工具

Yellowdog Update Modifier 一种基于rpm的包管理工具。
1、支持通配符安装软件包

仓库列表

yum repolist all | enabled | disabled
显示所有仓库 | 可用仓库 | 不可用仓库

软件包

yum list [all] | available | updates | installed php*
1\显示所有php开头的软件包 | php开头可用的 | php开头可更新的 | php开头已安装的
yum list recent
2\显示仓库中最近增加的软件包
yum info tree
3\查询软件包信息
yum provides /bin/mv
4\查看文件的来源
yum install | update | remove tree
5\安装|升级|卸载程序包

包组相关的命令

yum groupinstall # 安装包组
yum groupupdate #更新包组
yum grouplist #显示包组
yum groupremove #移除包组
yum groupinfo #查看包组信息

Yum 缓存管理

yum clean all
1、清理本地缓存
yum clean plugins
2、清除插件缓存
yum makecache
3、构建缓存

其他

yum search php #搜索
yum deplist php #查看指定包所依赖的capabilities
yum history #查看yum事务历史

下载包

在某些情况下，我们会需要只下载rpm软件安装包，而不是直接安装到本机上。
yum包管理工具通过安装yum-downloadonly扩展包，使yum可以只下载安装包。
使用参数
–downloadonly #指定只下载，不按照
–downloaddir=/pathtodir #指定下载目录
示例
yum install vim --downloadonly --downloaddir=/tmp

通过动态链接库和模块查找所属包

yum whatprovides libstdc++.so.6
查找动态链接库libstdc++的信息。从中找到所属的安装包，使用yum 安装
yum install libstdc++-4.4.7-3.el6.i686
