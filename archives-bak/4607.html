Docker 命令使用指南
admin 2018/11/27 2018-11-27

docker 有众多的子命令，用于管理不同的资源和服务类型。
为了更好的学习Docker服务的管理，下面对docker子命令进行分类学习。

子命令类别 子命令
环境信息 info 显示全系统信息

version
容器生命周期管理

create 创建容器

exec 在正在运行的容器上运行命令

kill  rm run

start/stop/restart 启动、停止、重启

pause/unpause 暂停/恢复容器中进程
镜像仓库命令

login 登录到Docker注册服务器

logout pull push search
镜像管理

build 通过dockerfile创建镜像

images 列表镜像

import 从归档文件中创建镜像

load 从TAR存档中加载镜像

rmi save tag

commit 通过容器创建新镜像
容器运维操作

attach 附在正在运行的容器上

export 将文件系统作为一个tar归档文件导出到STDOUT

inspect 获取容器/镜像的元数据

port ps rename stats top wait

cp 将文件(夹)从容器拷贝到主机

diff 检查容器文件系统上的更改

update
容器资源操作 volume network
系统日志信息

events 从服务器获取实时时间

history 显示镜像的历史

logs
