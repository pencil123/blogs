python unicode 编码的转换
admin 2016/09/01 2017-08-22

unicode是一种包含了世界上所有的字符的编码格式；同时它因为没有定义存储方式和不同字符之间占用的字节不同。在实际使用上并不普及，所以在编程中需要将unicode转化为其他编码格式；中文：如utf-8,GBK
编码 String encode() Method
The method encode() returns an encoded version of the string. Default encoding is the current default string encoding. The errors may be given to set a different error handling scheme.
str.encode(encoding=’UTF-8′,errors=’strict’)
例子：
>>> s = u’中国’
>>> type(s)
<type ‘unicode’>
>>> s
u’\u4e2d\u56fd’
>>> s = s.encode(‘utf-8’)
>>> s
‘\xe4\xb8\xad\xe5\x9b\xbd’
>>> type(s)
<type ‘str’>

解码：String decode() Method
The method decode() decodes the string using the codec registered for encoding. It defaults to the default string encoding.
Str.decode(encoding=’UTF-8′,errors=’strict’)
例子：
>>> type(s)
<type ‘str’>
>>> s
‘\xe4\xb8\xad\xe5\x9b\xbd’
>>> s = s.decode(‘utf-8′)
>>> s
u’\u4e2d\u56fd’
>>> type(s)
<type ‘unicode’>

Unicode字符串可以用多种方式编码为普通字符串, 依照你所选择的编码(encoding):

#将Unicode转换成普通的Python字符串:”编码(encode)”
unicodestring = u”Hello world”
utf8string = unicodestring.encode(“utf-8”)
asciistring = unicodestring.encode(“ascii”)
isostring = unicodestring.encode(“ISO-8859-1”)
utf16string = unicodestring.encode(“utf-16”)

#将普通的Python字符串转换成Unicode: “解码(decode)”
plainstring1 = unicode(utf8string, “utf-8”)
plainstring2 = unicode(asciistring, “ascii”)
plainstring3 = unicode(isostring, “ISO-8859-1”)
plainstring4 = unicode(utf16string, “utf-16”)

plainstring1==plainstring2==plainstring3==plainstring4
