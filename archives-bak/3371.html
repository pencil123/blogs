html 页面
admin 2018/01/31 2018-03-26

html 页面编写过程中有一些元素需要特别注意使用方法。

!DOCTYPE 声明

此声明必须在html文档的第一行，位于html标签之前
此声明不是 HTML 标签；它是指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。
在 HTML 4.01 中，<!DOCTYPE> 声明引用 DTD，因为 HTML 4.01 基于 SGML。DTD 规定了标记语言的规则，这样浏览器才能正确地呈现内容。
HTML5 不基于 SGML，所以不需要引用 DTD。声明的写法<!DOCTYPE html>

head 标签

所有头部元素的容器；可包含脚本，指示浏览器在何处可以找到样式表，提供元信息，等等。
包括的标签：title、base、link、meta、script和style。

title 标签

定义文档的标题。

base 标签

为页面上的所有链接规定默认地址或默认目标（target）
默认地址：页面链接使用相对地址
默认目标：控制页面打开方式

link 标签

定义文档与外部资源的关系。常用于链接样式表

href 规定被链接文档的位置
type 规定被链接文档的 MIME 类型
rel 规定当前文档与被链接文档之间的关系
rel:stylesheet 文档的外部样式表
rel:start 集合中的第一个文档
rel:next 集合中的下一个文档
rel:prev 集合中的上一个文档
rel:icon 图标favicon.ico
rel:profile 链接标识的映射关系
rel:alternate 提供链接到文档的替代版本
rel:index 文档的索引

meta 标签

元数据（metadata）是关于数据的信息
meta 元素被用于规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据。

charset 页面的编码格式
content 与 http-equiv 或 name 属性相关的元信息
http-equiv 把 content 属性关联到 HTTP 头部
http-equiv:refresh 定义文档自行刷新的时间间隔
http-equiv:content-type 指定文档的字符编码
name 把 content 属性关联到一个名称。
name:author 作者
name:description 描述
name:keywords 关键词
name:viewport 控制视口（用户的网页可见区域）

style 标签

为 HTML 文档定义样式信息。

script 标签

定义客户端脚本，比如 JavaScript。
