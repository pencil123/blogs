MongoDB数据更新
admin 2016/11/16 2018-03-29

db.collection.update( criteria, objNew, upsert, multi )

update()函数接受以下四个参数：

criteria update的查询条件，类似sql update查询内where后面的
objNew update的对象和一些更新的操作符（如$,$inc…）等，也可以理解为sql update查询内set后面的
upsert 这个参数的意思是，如果不存在update的记录，是否插入objNew,true为插入，默认是false，不插入
multi mongodb默认是false,只更新找到的第一条记录，如果这个参数为true,就把按条件查出来多条记录全部更新

更新操作

更新操作定位于一个集合。通过过滤器定位到要更新文档，过滤器的使用方法和查询的方法相同。

字段操作：

$currentDate 字段值设为当前时间；
Sets the value of a field to current date, either as a Date or a Timestamp.
$inc 字段值增加指定数量；
Increments the value of the field by the specified amount.
$min 比较给定值和存储值；取小的
Only updates the field if the specified value is less than the existing field value.
$max 比较给定值和存储值；取大的
Only updates the field if the specified value is greater than the existing field value.
$mul 字段值乘以指定值；
Multiplies the value of the field by the specified amount.
$rename 重命名；Renames a field.
$set 设置新值；Sets the value of a field in a document.
$setOnInsert 如果更新导致文档插入，则设置字段值；
Sets the value of a field if an update results in an insert of a document. Has no effect on update operations that modify existing documents.
$unset 删除字段；
Removes the specified field from a document.

例子：
db.user.update({“name”:”james”},{$set:{“addr”:”China”},$currentDate:{lastModified:true}})
名字为james #过滤器；更改addr为China,修改最后修改时间为当前时间

数组操作

$ Acts as a placeholder to update the first element that matches the query condition in an update.
$addToSet Adds elements to an array only if they do not already exist in the set.
$pop 删除数组的第一个或最后一个项目
Removes the first or last item of an array.
$pull 删除与指定查询匹配的所有数组元素
Removes all array elements that match a specified query.
$pushAll Deprecated. Adds several items to an array.
$push Adds an item to an array.
$pullAll Removes all matching values from an array.

修饰符Modifiers

$each Modifies the $push and $addToSet operators to append multiple items for array updates.
$position Modifies the $push operator to specify the position in the array to add elements.
$slice Modifies the $push operator to limit the size of updated arrays.
$sort Modifies the $push operator to reorder documents stored in an array.

二进制

$bit Performs bitwise AND, OR, and XOR updates of integer values.

隔离

$isolated Modifies the behavior of a write operation to increase the isolation of the operation.
