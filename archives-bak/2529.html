logstash源码安装时的目录结构

home 安装目录 –
bin 执行文件 –
settings 配置文件 path.settings
logs 日志 path.logs
plugins 非Ruby-Gem插件文件 path.plugins
data 数据文件 path.data

配置文件
设置文件：用于设置logstash启动和执行的。config目录

logstash.yml:控制logstash的运行，如管道设置，配置文件的位置，日志记录选项和其他设置。
两种配置方式：分层配置和平键配置

Node identity node.name
Data path path.data
Pipeline Settings pipeline.workers\pipeline.output.workers\pipeline.batch.size….
Pipeline Configuration Settings path.config\config.string\config.debug….
Module Settings modules…
Queuing Settings queue.type\queue.page_capacity…
Dead-Letter Queue Settings dead_letter_queue.enable….
Metrics Settings http.host
Debugging Settings log.levelinfo\path.logs

jvm.options：Contains JVM configuration flags
startup.options：

log4j2.properties：日志

管道配置文件：设置input，output，filter，codec等执行过程的文件。
