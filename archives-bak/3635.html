wordpress 内容页(single)添加上一帖和下帖导航

get_the_post_navigation( array $args = array() )

生成上一贴/下一贴的导航

参数：数组形式；可选；使用默认参数

‘prev_text’ (string) 上一贴显示的内容 Default ‘%title’.
‘next_text’ (string) 下一贴显示的内容. Default ‘%title’.
‘in_same_term’ (bool) 链接是否应该处于同一分类术语中. Default false.
‘excluded_terms’ (array|string) 排除的术语ID的数组或逗号分隔列表
‘taxonomy’ (string) 分类法, if $in_same_term is true. Default ‘category’.
‘screen_reader_text’ (string) 导航元素的屏幕阅读器文本. Default ‘Post navigation’.
