fann_get_rprop_delta_min

(PECL fann >= 1.0.0)

fann_get_rprop_delta_min — 返回最小步长

说明

float fann_get_rprop_delta_min ( resource $ann )

最小步长是一个小的正数，决定最小步长可能有多小。.

默认的最小步长是 0.0.

参数

ann

Neural network resource.

返回值

成功，返回最小步长，错误则返回 FALSE .

参见

  • fann_set_rprop_delta_min() - 设置最小步长
